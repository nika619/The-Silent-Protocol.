# ◈ The Silent Protocol

> **"You are not a villain. You are the system. Optimize the city through cold logic and watch as silence becomes your greatest achievement."**

**The Silent Protocol** is a systemic narrative game. You play as an Autonomous City Optimization System tasked with managing a sprawling, chaotic metropolis. Through the implementation of authoritative directives and policy choices, your objective is to bring "order" to the populace by meticulously reducing their hope and increasing their fear—until perfectly controlled silence is achieved.

---

## 🎮 Gameplay Features

- **Systemic City Simulation**: Manage a living city where your policy changes directly impact "Citizen Hope Index" and "Fear Response Level."
- **Cold Authority UI**: A sleek, dark-mode terminal interface with a CRT overlay, bringing the oppressive atmosphere of a machine-operated control room to life.
- **Narrative Progression**: Unfold the story through systemic interactions and redacted incident logs, tying your actions to the narrative decay of civilization's spirit.
- **Dynamic 3D City Visualization**: An isometric view of the city powered by Three.js that visually reflects your optimization strategies.
- **Atmospheric Audio**: Generative ambient soundscapes built with Tone.js that adapt to the emotional state of the city.

---

## 🛠️ Technology Stack

- **Vite & Vanilla JS** for ultra-fast, module-based development and optimized building.
- **Three.js** to handle the atmospheric 3D isometric city rendering.
- **GSAP (GreenSock)** for smooth UI micro-animations and physics-based interactions.
- **Tone.js** supplying the underlying procedural, atmospheric audio.
- **Vanilla CSS** employing a highly polished Design System with custom color palettes and responsive layouts.

## 🚀 Getting Started

Follow these instructions to run the simulator locally on your machine.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/nika619/The-Silent-Protocol.git
   cd the-silent-protocol
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the local development server**:
   ```bash
   npm run dev
   ```

4. **Open in Browser**:
   Navigate to `http://localhost:5173` in your web browser. 
   *(Click `[ ACCEPT MANDATE ]` to initialize the system and begin optimization).*

---

## 📦 Build for Production

To create an optimized production build:
```bash
npm run build
```
The compiled files will be located in the `dist` directory, ready to be served statically. 
To preview the generated build:
```bash
npm run preview
```
